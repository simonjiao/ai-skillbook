# Taxonomy

Use these categories for `type` in front matter:

- `skill` - reusable agent behavior or task-specific capability.
- `plugin` - bundled capability, connector, MCP server, or app integration.
- `tool` - CLI, API, library, service, or local utility.
- `prompt` - reusable prompt pattern or prompt-review note.
- `workflow` - multi-step operating procedure.
- `automation` - scheduled or recurring task.
- `case-study` - concrete usage report with evidence.
- `reference` - stable background material.

Common statuses:

- `draft`
- `tested`
- `adopted`
- `deprecated`
